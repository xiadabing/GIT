1 类 = 属性+方法
属性 指特征
方法 指的是一个行为

2 对象，是类的具体实现
对象（实例）= 实例属性+实例方法
def __init__(self)
def 方法名（self）

3 访问实例属性和实例方法
在类外 对象.实例属性   对象.实例方法
在类例 self.实例属性  self.实例方法

创建实例属性：
在类外 对象.实例属性 = 属性值  不推荐
在类里 self.实例属性 = 属性值

3 类 是一类事物的抽象方法
类 = 类属性+类方法
类名.实例属性  不行
类名.实例方法（self）  不行

类.类属性  对象.类属性
类名.类方法（cls） 对象.类方法（cls）

静态方法：

类继承：子类 父类
子类拥有父类的所有方法，私有方法除外
重写：覆盖父类中的某个方法，子类中定义的方法要与父类中的方法名一致
拓展：在父类的基础上添加一些功能
super.()父类中的方法

自动化测试流程：
1 需求分析
2 写测试用例
3 执行测试用例
4 报表，总结

单元测试：
